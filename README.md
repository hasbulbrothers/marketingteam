# HB Marketing Task Manager

Minimal internal task management system for the marketing team.

## Stack

- Next.js App Router
- Tailwind CSS
- shadcn/ui
- Convex
- Clerk

## Local setup

1. Copy `.env.example` to `.env.local`
2. Add Clerk and Convex keys
3. Run `npm install`
4. Run `npm run dev`

The UI includes a preview mode when auth or backend keys are not configured yet.
